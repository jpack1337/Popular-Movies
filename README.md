# Popular-Movies
Struggling to find something to watch? Popular Movies is your answer! Utilizing TheMovieDB.org API, the app makes use of the following endpoints:
- <b>top_rated</b>
- <b>now_playing</b>
- <b>popular</b>
- <b>upcoming</b>
